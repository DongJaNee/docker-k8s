# WSL(Windows Subsystem for Linux)이란?
## 개요
윈도우환경에서 리눅스를 사용 
WSL을 이용하면 듀얼부팅이나 VM과 같은 느린 환경이 아니라 윈도우에서 리눅스 환경처럼 Powershell을 Bash처럼 사용하고, Linux 명령어를 사용할 수 있으며 Linux커널 이용 가능
